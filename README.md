# Factorización LU - Excel y Web

## 📊 Archivo Excel
El archivo `jueves LU.xlsx` contiene:
- Factorización LU paso a paso (método Doolittle)
- Matrices u0, u1, u2, u3
- Multiplicadores f21, f31, f32
- Resolución del sistema L·d = b y U·x = d

### Sistemas resueltos:
| Sistema | Solución |
|---------|----------|
| 8x₁ + 4x₂ + x₃ = 11<br>2x₁ + 5x₂ + 4x₃ = 4<br>2x₁ - x₂ + 6x₃ = 7 | x₁ = 1.4385<br>x₂ = -0.2869<br>x₃ = 0.6393 |
| 10x₁ + 2x₂ - x₃ = 27<br>-3x₁ - 6x₂ + 2x₃ = -61.5<br>x₁ + x₂ + 5x₃ = -21.5 | x₁ = 0.5<br>x₂ = 8<br>x₃ = -6 |

## 🌐 Página Web
La página `prueba.html` permite:
- Resolver sistemas hasta 6×6
- Elegir método: Doolittle, Crout, Cholesky
- Visualizar paso a paso similar al Excel
- Ver las fórmulas Excel utilizadas
